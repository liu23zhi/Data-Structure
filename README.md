# Data-Structure
这是一个关于数据结构的项目，重点研究数组、链表、栈、队列、树和图等常见数据结构的设计、实现与应用。其目的是加深对数据结构基本概念的理解，掌握各类数据结构的核心操作（插入、删除、查找和遍历），并运用相关知识解决实际编程问题。

---

## 项目结构

```
.
├── Makefile
├── part1_block_search/
│   └── block_search.c   # 分块查找
├── part2_graph_search/
│   └── graph_search.c   # 有向图路径搜索
└── part3_sparse_matrix/
    └── sparse_matrix.c  # 稀疏矩阵转置
```

## 编译与运行

```bash
# 编译全部
make all

# 分别运行三个部分
make run1   # 分块查找
make run2   # 有向图路径搜索
make run3   # 稀疏矩阵转置

# 清除编译产物
make clean
```

## 第一部分 – 分块查找 (`part1_block_search/block_search.c`)

**顺序表**: `8, 14, 6, 9, 10, 22, 34, 18, 19, 31, 40, 38, 54, 66, 46, 71, 78, 68, 80, 85, 100, 94, 88, 96, 87`  
**块长** = 5，**块数** = 5，**查找目标** = 46

| 功能 | 说明 |
|------|------|
| 自动划分块并生成索引表 | 扫描每块求最大值，建立 `(maxKey, start)` 索引 |
| 索引顺序查找 | 顺序遍历索引表定位块，再顺序查找块内元素 |
| 索引折半查找 | 二分搜索索引表定位块，再顺序查找块内元素 |
| 链式块存储与查找 | 每块用单链表存储，索引顺序定位块后遍历链表 |

每一步均有 ASCII 图形化展示（顺序表分块示意、索引表、链表结构）。

## 第二部分 – 有向图路径搜索 (`part2_graph_search/graph_search.c`)

**有向图 G**（7 个顶点）边集：  
`5->1, 5->2, 5->3, 1->2, 1->4, 3->1, 3->4, 4->2, 4->6, 6->2`

| 功能 | 说明 |
|------|------|
| 邻接矩阵存储 | 打印完整的 7×7 邻接矩阵 |
| 邻接表展示 | 列出每个顶点的出边 |
| DFS 所有简单路径 | 从顶点 5 到顶点 2 的全部简单路径（共 9 条） |
| DFS 长度为 3 的路径 | 剪枝 DFS 只输出步数等于 3 的路径（共 3 条） |
| BFS 最短路径（队列） | 层次遍历求最短路径，迭代回溯打印路径 |

## 第三部分 – 稀疏矩阵转置 (`part3_sparse_matrix/sparse_matrix.c`)

**示例矩阵 A**（6 行 × 7 列，8 个非零元素）

| 算法 | 时间复杂度 | 辅助空间 |
|------|-----------|---------|
| 普通转置 | O(cols × t) | O(1) |
| 快速转置（colCount + colStart） | O(cols + t) | O(2 × cols) |
| 优化快速转置（单辅助数组） | O(cols + t) | O(cols) |

每种算法均输出三元组表、稠密矩阵形式及实际操作次数对比。
